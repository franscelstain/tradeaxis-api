# DAILY PIPELINE

## Command
market-data:daily

## Signature
php artisan market-data:daily 
  {--requested_date=} 
  {--source_mode=} 
  {--correction_id=} 
  {--latest}

## Fungsi
Menjalankan seluruh pipeline EOD dalam satu entry point.

## Opsi
- --requested_date=
- --source_mode=
- --correction_id=
- --latest
